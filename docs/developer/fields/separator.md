# Separator Field (`type: separator`)

UI helper field for spacing or horizontal rule.

- Use `line: true` for `<hr>`.
- Use `size` for spacing utility value.

```php
'spacer' => [
    'type' => 'separator',
    'line' => true,
    'size' => 2,
]
```

## Additional Examples

```php
'spacer' => [
    'type' => 'separator',
    'line' => false,
    'size' => 3,
]
```
