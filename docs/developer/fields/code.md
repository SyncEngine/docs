# Code Field (`type: code`)

CodeMirror editor with optional language extension and tag insertion.

## Typical Props

- `language`
- `basicSetup`
- `taggable`
- `contained`

## Example

```php
'template' => [
    'label' => $this->trans('Template'),
    'type' => 'code',
    'taggable' => true,
]
```
