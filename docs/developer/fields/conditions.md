# Conditions Field (`type: conditions`)

Grid-based condition builder with source/key/operator/compare columns.

## Typical Props

- `taggable`
- `sortable`
- `conditionTypes`
- optional `source` config

## Example

```php
'conditions' => [
    'label' => $this->trans('Conditions'),
    'type' => 'conditions',
    'sortable' => true,
]
```
